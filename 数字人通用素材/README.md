# 数字人通用素材包

本目录用于数字人原型、口型同步、UI 占位和场景合成测试。

## 已包含

- `01_人物图片`：5 张已下载的高清中国/东亚女性人像，包括现代棚拍、自然微笑、时尚、北京光影和汉服造型。
- `02_人物视频`：CC0 正面说话视频、Pexels 对镜说话视频和 Mixkit 绿幕视频入口。
- `03_人物语音`：2 段可直接使用的 TTS WAV，以及 3 组公有领域普通话真人朗读来源。
- `04_背景素材`：CC0 办公场景和抽象背景入口。
- `05_授权与来源`：许可页、使用边界和来源说明。
- `素材清单.csv`：完整素材索引。

## 人物图推荐顺序

1. `01_精致棚拍_中国风女性_正脸.jpg`：颜值和画质最好，适合头像、封面及形象参考。
2. `05_汉服造型_白底正面.jpg`：适合国风、文旅和文化讲解数字人。
3. `04_北京艺术光影_正面人像.jpg`：适合艺术、内容和情绪类形象。
4. `02_亲和微笑_白衬衫_正脸.jpg`：适合客服、教育和讲解场景。
5. `03_现代时尚_红帽微笑.jpg`：适合直播、时尚和年轻化内容。

## 使用说明

- 图片已经下载到本地；同目录 `.url` 文件可打开原始素材页。
- 用于真人数字分身、声音克隆或产品代言前，必须另外取得明确的肖像和声音授权。
- Pexels 素材不得暗示画面人物认可或代言你的产品。
- 做数字人口型驱动时，正脸、闭嘴或自然微张、无遮挡的素材通常最稳定。
